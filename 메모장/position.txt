position 박스가 위치하는 방법

static (기본값) ==> 관련된 다른 박스들의 상태에 따른 위치

absolute => 자기 분모값의 위치값을 기주으로 자기 위치를 잡는다.
	단, 부모가 위치값을 갖고 있는 경우
	부모가 위치값이 없으면 body(view point)가 부모값
	위치값은 top left right bottom 
	z-index를 이용하여 상, 하를 조정

relative => static + absolute
          ==> margin이 다 통용 left top button right 사용가능

*주로 position: absolute 값을 갖는 부모는 position:relative를 준다.
	

fixed : 뷰포인트로 부터 고정된 값
